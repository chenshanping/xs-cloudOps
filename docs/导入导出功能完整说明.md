# 导入导出功能完整说明

## 功能概述

代码生成器已完整实现Excel导入导出功能，包括：

### 后端功能
- ✅ Excel导出：将数据库记录导出为Excel文件
- ✅ Excel导入：从Excel文件批量导入数据
- ✅ 模板下载：下载带表头的Excel模板
- ✅ 数据转换：自动处理数据字典、关联关系、开关字段、时间格式

- ✅ 操作日志：记录导入导出操作（不记录二进制内容）

### 前端功能
- ✅ 导出按钮：点击导出当前筛选条件下的数据
- ✅ 导入按钮：选择Excel文件上传导入
- ✅ 下载模板按钮：下载导入模板
- ✅ 导入结果提示：显示成功/失败数量和错误详情
- ✅ 权限控制：使用 `v-permission` 指令控制按钮显示

## API接口说明

### 导出接口

```
GET /api/v1/{module}/export
```

**请求参数：** 与列表查询相同，支持所有筛选条件

**响应：** Excel文件流（application/vnd.openxmlformats-officedocument.spreadsheetml.sheet）

**示例：**
```bash
curl -X GET "http://localhost:8080/api/v1/productType/export?status=1" \
  -H "Authorization: Bearer {token}" \
  --output productType.xlsx
```

### 导入接口

```
POST /api/v1/{module}/import
Content-Type: multipart/form-data
```

**请求参数：**
- `file`: Excel文件（.xlsx 或 .xls）

**响应：**
```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "success_count": 10,
    "fail_count": 2,
    "errors": [
      "第3行: 状态值无效",
      "第5行: 产品类型不存在"
    ]
  }
}
```

### 下载模板接口

```
GET /api/v1/{module}/template
```

**响应：** Excel模板文件

**示例：**
```bash
curl -X GET "http://localhost:8080/api/v1/productType/template" \
  -H "Authorization: Bearer {token}" \
  --output template.xlsx
```

## 错误处理

### 导入错误类型

1. **文件格式错误**
   - 错误：不支持的文件格式
   - 解决：只支持 .xlsx 和 .xls 格式

2. **表头不匹配**
   - 错误：Excel表头与模板不一致
   - 解决：使用"下载模板"获取正确的模板

3. **数据验证失败**
   - 错误：第X行: 字段值无效
   - 解决：检查该行数据是否符合要求

4. **数据字典值不存在**
   - 错误：第X行: 状态值"xxx"不存在
   - 解决：使用模板中的下拉选项值

5. **关联对象不存在**
   - 错误：第X行: 找不到产品类型"xxx"
   - 解决：确保关联对象已存在

### 前端错误提示

导入完成后会显示详细结果：

```
导入完成
成功: 10 条，失败: 2 条

错误详情:
第3行: 状态值"已停用"不存在，请使用: 启用, 禁用
第5行: 找不到产品类型"测试类型"
```

## 常见问题

### Q1: 导入时提示"文件太大"

**原因：** 超过了文件上传大小限制

**解决：** 修改配置文件中的上传限制
```yaml
# config.yaml
server:
  max_upload_size: 10  # MB
```

### Q2: 导出的Excel中文乱码

**原因：** Excel编码问题

**解决：** 使用UTF-8 BOM编码，已在代码中处理

### Q3: 导入后关联字段为空

**原因：** Excel中填写的关联对象名称不存在

**解决：** 
1. 确保关联对象已创建
2. 名称必须完全匹配（区分大小写）
3. 使用模板中的示例数据

### Q4: 按钮不显示

**原因：** 权限未配置

**解决：** 参考 [QUICK_FIX_导入导出按钮不显示.md](./QUICK_FIX_导入导出按钮不显示.md)
