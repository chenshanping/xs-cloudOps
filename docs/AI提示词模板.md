# AI 提示词模板 - 项目需求文档与数据库设计

```bash
我想开发一个诗歌管理系统，请帮我完成以下工作：

## 一、系统命名
请帮我设计：
- 系统中文名（2-5个字，简洁有意义）
- 系统英文名（驼峰命名，适合做项目名）
- login_slogan（登录页标语，10-15字）
- login_desc（系统描述，30-50字）
- login_features（4个核心特性，每个2-4字）

## 二、功能需求
系统包含以下角色和功能：

### 角色1：管理员


### 角色2： 诗歌管理员

### 角色3：用户
## 三、输出要求
1. 需求文档（Markdown格式）
   - 项目概述
   - 用户角色说明
   - 功能模块详细说明
   - 数据字典
   - API接口规划

2. 数据库设计（MySQL SQL脚本）
   - 完整的建表语句
   - 字段注释
   - 索引设计
   - 初始化数据
3.项目有内置代码生成器 结合来完成 最后一部分需要写代码生成器如何使用生成业务模块
4.所有文件生成到一个md文件里面
```
