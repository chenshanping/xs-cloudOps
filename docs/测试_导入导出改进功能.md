## 常见问题排查

### 问题1：下载模板没有批注

**检查：**
1. 是否重新生成了代码？
2. Excel是否支持批注？
3. Excel设置是否显示批注？

**解决：**
```
Excel选项 -> 高级 -> 显示 -> 批注和指示器 -> 仅指示器和批注
```

### 问题2：下拉验证不显示

**检查：**
1. 是否点击了正确的单元格（第2行开始）？
2. 数据字典是否存在？
3. 数据字典是否有数据？

**验证：**
```sql
SELECT * FROM sys_dict_data WHERE dict_type = 'common_status';
```

### 问题3：导入时没有验证

**检查：**
1. 是否重新生成了代码？
2. 后端代码是否更新？
3. 是否重启了服务器？

**验证：**
查看 `go-base-server/api/v1/productType.go` 中的 `ImportProductType` 方法，应该包含严格验证逻辑。

### 问题4：错误信息不显示有效值

**检查：**
1. 数据字典是否存在？
2. `GetDictDataList` 方法是否添加？

**验证：**
```go
// 在 service/dict.go 中应该有
func (s *DictService) GetDictDataList(dictType string) ([]model.SysDictData, error)
```
