## 数据流程

### 导出流程

```
用户点击导出
    ↓
前端调用 export{Module} API
    ↓
后端查询数据（支持筛选）
    ↓
数据转换：
  - 数据字典：1 -> "启用"
  - 关联关系：123 -> "产品名称"
  - 开关字段：1 -> "是"
  - 时间格式化
    ↓
创建Excel文件
    ↓
返回文件流（blob）
    ↓
前端下载文件
    ↓
操作日志记录：[文件下载]
```

### 导入流程

```
用户选择Excel文件
    ↓
前端调用 import{Module} API
    ↓
后端解析Excel文件
    ↓
逐行验证：
  - 跳过空行
  - 数据字典值是否存在
  - 关联对象是否存在
  - 必填字段是否为空
  - 数据格式是否正确
    ↓
数据转换：
  - 数据字典："启用" -> 1
  - 关联关系："产品名称" -> 123
  - 开关字段："是" -> 1
  - 时间解析
    ↓
保存到数据库
    ↓
返回导入结果：
  - 成功数
  - 失败数
  - 错误详情
    ↓
前端显示结果
    ↓
操作日志记录：[文件上传] + 导入结果
```

## 关键代码

### Excel工具类

```go
// 创建导出器
exporter := utils.NewExcelExporter("产品类型")

// 设置表头和批注
exporter.SetHeadersWithComments(headers, comments)

// 添加下拉验证
exporter.AddDataValidation(colIndex, options, 2, 1000)

// 添加数据行
exporter.AddRow(values)

// 生成文件
buffer, _ := exporter.SaveToBuffer()
```

### 数据字典转换

```go
// 导出：值 -> 标签
label := service.Dict.GetDictLabel("common_status", "1") // "启用"

// 导入：标签 -> 值
value := service.Dict.GetDictValue("common_status", "启用") // "1"

// 获取所有选项（用于验证）
list, _ := service.Dict.GetDictDataList("common_status")
```

### 导入验证

```go
// 验证数据字典值
dictValue := service.Dict.GetDictValue("common_status", cellValue)
if dictValue == "" {
    // 获取有效值列表
    dictList, _ := service.Dict.GetDictDataList("common_status")
    validValues := make([]string, 0, len(dictList))
    for _, item := range dictList {
        validValues = append(validValues, item.Label)
    }
    // 记录错误
    rowErrors = append(rowErrors, fmt.Sprintf(
        "状态值\"%s\"不存在，有效值: %s", 
        cellValue, strings.Join(validValues, ", ")
    ))
    hasError = true
}
```

## 错误处理

### 导入错误类型

| 错误类型 | 示例 | 处理 |
|---------|------|------|
| 数据字典值不存在 | 状态="已停用" | 拒绝导入，显示有效值 |
| 关联对象不存在 | 产品类型="测试" | 拒绝导入，提示先创建 |
| 必填字段为空 | 名称=(空) | 拒绝导入，提示必填 |
| 数据格式错误 | 价格="abc" | 拒绝导入，提示格式 |
| 开关值无效 | 是否推荐="可能" | 拒绝导入，显示有效值 |

### 错误信息格式

```
导入完成
成功: 8 条，失败: 2 条

错误详情:
第3行: 状态值"已停用"不存在，有效值: 启用, 禁用; 产品类型"测试"不存在，请先创建该产品类型
第5行: 价格为必填项; 是否推荐值"可能"无效，有效值: 是, 否
```
